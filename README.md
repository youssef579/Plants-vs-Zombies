# Plants Vs Zombies

## How to run

1. Clone the repo `git clone https://github.com/youssef579/Plants-vs-Zombies.git`
2. Build the project using [CMake](https://cmake.org/download/)

   ```bash
   cmake -B build
   cmake --build build
   ```

3. Run the game `./build/bin/main`
4. Check this [guide](https://www.sfml-dev.org/tutorials/3.0/getting-started/cmake/) for further details

## Project structure

- /src: Implementation files (.cpp)
- /include: Declaration files (.hpp)
- /assets: Resources (images, fonts and sounds)
